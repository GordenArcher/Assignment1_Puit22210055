
# Iris Dataset ML Classification

Machine Learning model using Logistic Regression to classify Iris flower species.

## Features
- Data exploration and visualization
- Logistic Regression model training
- Model evaluation with metrics
- Prediction on new data

## Installation
\`\`\`bash
pip install -r requirements.txt
\`\`\`

## Usage
\`\`\`bash
python main.py
\`\`\`

## Results
- Achieves 95-100% accuracy
- Classifies 3 species: Setosa, Versicolor, Virginica
- Includes visualizations and performance metrics
